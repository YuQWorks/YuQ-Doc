# Yu-Core

YuQ 框架核心依赖。

# 特性

  • 依赖注入
      YuQ 实现了部分 JSR-330 的特性，后续会支持完整的 JSR-330。
  • 代码注入
      YuQ 允许在应用加载时对类注入指定代码。
  • 方法拦截
      YuQ 提供在方法被调用时（调用前，调用后，异常时）进行操作，允许取消方法执行，修改返回值，修改异常信息等操作。
  • 参数验证
      YuQ 提供在任意方法被调用前的参数验证，支持 JSR-349 规范。
  • 运行时重载
      YuQ 提供应用热重载以满足某些情况下重启不便进而导致的开发不便。
